# 무무 시티 익스프레스

실차 비율의 스포츠카를 성장시키며 산악 기지에서 해안 도시까지 달리는 3인칭 3D 배달 레이싱 게임입니다.

## 핵심 플레이

- 차체 색상, 바퀴, 지붕 장식, 스티커 조합
- 직사각형 격자를 제거한 곡선 순환도로·방사형 대로·비정형 골목
- 대각선으로 흐르는 강과 강을 건너는 3개의 선택형 다리
- 도로 방향에 맞춰 회전하며 불규칙하게 배치되는 220여 개 도시 건물
- 실제 도로와 열린 다리를 따라 계산되는 미니맵 경로와 차량 위 방향 내비
- 다음 교차로의 직진·좌회전·우회전·유턴 안내
- 학교, 도서관, 공원, 박물관, 관측소 배송 미션
- 배송본부, 센트럴 광장, 카페, 놀이터, 마을정원, 컬러마켓, 테크캠퍼스, 푸드마켓
- 스로틀·브레이크·조향 관성과 부드러운 추적 카메라
- 왼쪽 마우스 시점 회전과 `시점 원위치` 버튼
- 아침·낮·노을·밤으로 이어지는 도시 시간 흐름
- 별 토큰 수집과 터보 주행
- 배송 도착 후 사자성어 암호 퀴즈
- 키보드와 터치 조작 지원

## 실행

```bash
npm install
npm run dev
```

## Meshy 실차 에셋 생성

채팅에 노출되지 않은 새 Meshy 키를 `.env.local`에만 넣습니다.

```dotenv
MESHY_API_KEY=새로_발급한_키
```

한 대만 먼저 생성해서 품질을 확인하거나, 확정된 5대를 순서대로 생성할 수 있습니다.

```bash
npm run meshy:vehicles -- --vehicle=snowbug
npm run meshy:vehicles -- --all
```

생성 결과는 `public/models/vehicles/*.glb`에 저장되고 `manifest.json`에 등록됩니다. 게임은 등록된 GLB를 자동으로 읽고, 아직 생성되지 않은 차량에는 내장 3D 차량을 사용합니다.

## 조작

- 이동: 방향키 또는 `WASD`
- 터보: `Space` 또는 `Shift`
- 시점 회전: 왼쪽 마우스 드래그
- 시점 복귀: 우측 상단 `시점 원위치`
- 퀴즈: 숫자 `1`~`4` 또는 화면 버튼

## 기술 스택

- React 19
- Vite 7
- Three.js

도시와 목적지 표식은 코드로 생성되며, 차량은 내장 3D 모델과 Meshy GLB 에셋을 자동으로 전환합니다.
